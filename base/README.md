# Css

`css`： Cascading Style Sheets -- 层叠样式表。<br>

!>为啥叫 `层叠样式表`？<br>
我大概的理解就是：当样式作用在同一个目标上时，样式可以通过 `叠加` ， `覆盖` 来进行修饰，子元素可以通过 `继承` 来表现。<br>

`css` 的出现，可以让我们的结构和表现分离，可以更灵活的修饰页面。

随着 `css3` 的发布，`css` 被依据功能分解成了模块，所以往后的 `css` 升级更多的是模块的升级，目前还不会有 `css4` 的说法 。<br>
